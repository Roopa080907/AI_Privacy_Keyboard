# 🔐 AI Privacy Keyboard

A smart web-based keyboard that detects sensitive information in real-time and alerts users before sending messages.

## 🚀 Features

* 📧 Detects Email Addresses
* 📱 Detects Phone Numbers
* 🔑 Detects OTP / Password keywords
* 🚨 Risk Level Indicator (Low, Medium, High)
* 📊 Visual Risk Meter
* 🔒 Generates Safe Masked Message
* ⚡ Real-time Analysis

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 💡 How It Works

The system analyzes user input using pattern matching (Regex) to detect sensitive information. Based on detected data, it assigns a risk score and alerts the user visually.
